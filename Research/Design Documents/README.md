# Design Documents

This folder contains written design documents exploring concepts,
architecture, and AI features for the REFLECT project.

These documents were used to:
- Think through feature feasibility
- Align technical choices with project goals
- Support portfolio and assessment documentation

Documents cover topics such as:
- Transcription
- Topic segmentation
- Sentiment analysis
- Semantic search
- PII detection
- Overall system concepts

Not all designs were fully implemented.
Some documents represent exploratory or comparative research only.
