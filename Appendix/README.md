# REFLECT — Project Pack / Appendix

This repository contains the appendix materials for **REFLECT**, an AI-powered journaling application developed as part of the Interaction Design programme at Fontys Eindhoven.

---

## How to view

The full appendix is published as an interactive Notion document and is best viewed there:

**[Open in Notion](https://www.notion.so/Inspiration-REFLECT-34369d42631b806d91cee08ad62ea31c)**

Notion renders all sections with proper formatting and embedded images. This is the recommended way to explore the materials.

If you can't access Notion or prefer an offline version, an exported PDF is included in this repository:

**[`REFLECT_Appendix.pdf`](./REFLECT_Appendix.pdf)**

---

## What's inside

The appendix covers:

- **Research** — Survey results, key insights, and the current AI prompt
- **Product Context** — System overview, backend capabilities, and design constraints
- **Inspiration** — Reference apps with annotated likes/dislikes
- **Starting Point** — A rough page structure *(not mandatory — provided as a reference only)*

---

## Note

This appendix is a supporting document for the REFLECT design brief. It is intended as context and inspiration for the design team, not a specification to follow literally.
